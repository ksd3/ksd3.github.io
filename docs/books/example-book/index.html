Preface

This is an example Quarto book with Typst PDF output.

About This Book

This template demonstrates how to create a book using Quarto with both HTML and Typst PDF output formats.

How to Build

To build this book:

# HTML and PDF
quarto render books/example-book/

# HTML only
quarto render books/example-book/ --to html

# PDF only (requires Typst)
quarto render books/example-book/ --to typst

Prerequisites

  • Quarto installed
  • Typst installed for PDF generation
